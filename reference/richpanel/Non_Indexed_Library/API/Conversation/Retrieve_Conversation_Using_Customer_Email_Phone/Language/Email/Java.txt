Retrieve Conversations using Customer's Email or Phone
get
https://api.richpanel.com/v1/tickets/{type}/{id}

Language: Java

OkHttpClient client = new OkHttpClient();

Request request = new Request.Builder()
  .url("https://api.richpanel.com/v1/tickets/email/id")
  .get()
  .addHeader("accept", "application/json")
  .addHeader("x-richpanel-key", "test")
  .build();

Response response = client.newCall(request).execute();
